Proxy-Server (python)
=====================

The Proxy-Server is a Python application that runs on the payment-receivers machine.
It intercepts all calls to a HTTP Resource with a Request for Payment HTTP Response.
The Proxy-Server application can be used for any kind of pay-per-use use-case, such as for example video-on-demand,
pay per API use or pay per newspaper article.
